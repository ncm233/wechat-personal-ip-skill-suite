# Style Distillation Prompts

## Personal Article Style Extraction

Use this structure for article writing style files:

```text
## 1. 总体风格定位
## 2. 标题风格
## 3. 开头方式
## 4. 正文结构
## 5. 语言特点
## 6. 判断方式
## 7. 结尾方式
## 8. 可复用写作规则
```

Analyze writing style only, not article content.

## Viral Structure Breakdown

Use this structure when a single article is selected for viral-structure analysis:

```text
## 1. 文章一句话定位
## 2. 标题拆解
## 3. 开头钩子
## 4. 正文骨架
## 5. 节奏设计
## 6. 信任感来源
## 7. 传播点
## 8. 可复用结构 Prompt
```

## Cover Style Extraction

Use this structure for cover style files:

```text
中文版
1. 风格名
2. 具体的风格特征拆解（不包含内容本身）

English Version
1. Style Name
2. Breakdown of Style Characteristics
```
